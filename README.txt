== Description ==
This plugin is developed as a project to hand in to the KRITEK team. It sinserts the meta tags in the code therein into all front end pages

== Installation ==
== METHOD ONE ==
1. Upload the `david-brian-acme-plugin` folder to the `wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== METHOD TWO ==
1. Go to the add plugin option in your WP Dashboard
2. Click the `Upload Plugin` button
3. Upload the `david-brian-acme-plugin.zip` file and click install.
4. Procedd to click activate.

== Disabling ==
To disable the ACME Software Meta Tags plugin, follow these steps:
1. Go to the 'Plugins' page in your WordPress admin panel.
2. Locate the 'DAVID BRIAN ACME PLUGIN' plugin.
3. Click on the 'Deactivate' link below the plugin name.

== Deletion ==
To delete the ACME Software Meta Tags plugin, follow these steps:
1. Go to the 'Plugins' page in your WordPress admin panel.
2. Locate the 'DAVID BRIAN ACME PLUGIN' plugin.
3. Click on the 'Delete' link below the plugin name.
4. Confirm the deletion when prompted.
